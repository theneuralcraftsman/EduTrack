EduTrack (Learning Management System) : Developed a backend for an 
LMS using REST APIs for managing online courses. Implemented CRUD 
operations for course and user management, progress tracking, 
assessments, and certificate generation. Integrated Razorpay for payments, 
implemented JWT-based authentication and authorization, and set 
automated email notifications and SMS alerts using Twilio. 
Tech Stack : Java ,SpringBoot
